### Scrum Poker Application

#### For build and Run
 1. git clone https://github.com/seydataskin/scrum-poker.git
 2. Enter project folder ("**cd scrum-poker**")
 3. Run command "**yarn**" for package install  (Check how you can install [yarn](https://classic.yarnpkg.com/en/))
 4. Run project using "**yarn start**" command
